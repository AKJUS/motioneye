__motionEye.eo__ estas forko de __motionEye__ tradukita al esperanto kaj kun plurlingva subteno.  
La subtenataj lingvoj estas :   
* franca (français)
* angla (english)

* araba ( العربية )
* bengala (	বাংলা )
* germana (Deutsch)
* hispana (Español)
* hinda ( हिन्दी )
* itala (Italiano)
* japana ( 	日本語 )
* malaja ( بهاس ملايو )
* panĝaba (	ਪੰਜਾਬੀ )
* portugala (Português)
* rusa (русский язык)
* ĉina ( 中文 )
